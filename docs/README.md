## 문서 작성 가이드

새로운 기술 문서를 작성할 때는 다음을 포함해주세요:

1. **개요**: 문서의 목적과 범위
2. **목차**: 주요 섹션 링크
3. **예제 코드**: 실제 사용 가능한 코드 스니펫
4. **문제 해결**: 일반적인 문제와 해결 방법
5. **참고 자료**: 관련 문서 링크
